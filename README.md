# CityWeatherApp
•	Built a fully functional weather app in Kotlin integrating a live weather REST API to fetch and display real-time data
•	Implemented API calls using Retrofit, parsed JSON responses, and displayed dynamic weather info including temperature, humidity, and conditions
•	Designed clean, responsive UI using Android XML layouts with proper error handling for network failures
•	Published on GitHub demonstrating version control, clean code structure, and independent project delivery
